Truss by RyanD

Truss is a tool for importing data to Canto Cumulus from xml delivered on the web.

Truss is a tool for importing data to Canto Cumulus from xml delivered on the web. It currently supports hard-coded values specified internally, but shortly on the todo list is pulling those references out into a seperate configuration, canceling, progress bars, etc.

Dependencies

Canto Cumulus

Install

Not pretty currently, best bet is to copy the ejp from the dist folder into the necessary folder (Windows should be C:/Program Files/Canto/Cumulus Client/ejp/ or similar).
